# HLSL Shaders

HLSL shader source for the DirectX 11 renderer.

## Compiling

Requires the Windows SDK (`fxc.exe`):

```
fxc /T vs_5_0 /E VSMain /Fo cell_vs.cso cell.hlsl
fxc /T ps_5_0 /E PSMain /Fo cell_ps.cso cell.hlsl
```

The `.cso` files will be generated by the build system (HlslStep.zig) and
embedded at comptime via `@embedFile` in `cell_pipeline.zig`.

The `.cso` files are not committed to git - they are build artifacts.
