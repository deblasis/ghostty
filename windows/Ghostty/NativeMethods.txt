// xaml interop: ISwapChainPanelNative is hand-written. CsWin32
// generates an ISwapChainPanelNative under
// Windows.Win32.System.WinRT.Xaml with IID
// F92F19D2-3ADE-45A6-A20C-F6F1EA90554B, but the WinUI 3
// SwapChainPanel responds to QueryInterface for IID
// 63aad0b8-7c24-40ff-85a8-640d944cc325 (the legacy XAML interop
// IID). The two are NOT interchangeable - the Ghostty DX12
// renderer relies on the legacy IID and would silently break if
// we switched. The hand-written shim moves from [ComImport] to
// [GeneratedComInterface] to keep the AOT story intact while
// retaining the correct IID.
